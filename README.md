# India Runs Data & AI Challenge - Resume Ranking ML

## Overview
This project implements a machine learning pipeline for resume ranking as part of the India Runs Data & AI Challenge.

## Features
- Resume data preprocessing
- Feature extraction
- Candidate ranking using machine learning
- Model evaluation

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Project Structure
```
.
├── data/
├── models/
├── notebooks/
├── src/
├── requirements.txt
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the main script:

```bash
python main.py
```
